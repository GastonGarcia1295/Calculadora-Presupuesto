## ¿Qué es?

Herramienta interactiva para cotizar proyectos de desarrollo web. El cliente selecciona el tipo de proyecto, la cantidad de secciones y opciones adicionales, y obtiene un presupuesto detallado al instante — con conversión automática a dólares y posibilidad de descargarlo en PDF.

---

## Funcionalidades

- **Tipos de proyecto configurables** — Landing Page, E-Commerce, Aplicación Web, Dashboard y más
- **Slider de páginas/secciones** — ajuste dinámico de la cantidad de secciones incluidas
- **Opciones adicionales** — Entrega urgente (+30%), mantenimiento mensual, integración de pagos, entre otras
- **Conversión ARS/USD en tiempo real** — vía API de cotización del dólar blue
- **Presupuesto final instantáneo** — se recalcula automáticamente con cada cambio
- **Exportación a PDF** — descargá el presupuesto completo con un clic
- **Diseño responsive** — funciona en mobile, tablet y desktop

---

## Cómo funciona

1. Seleccionás el **tipo de proyecto** desde el menú desplegable
2. Ajustás la cantidad de **páginas o secciones** con el slider
3. Marcás las **opciones adicionales** que necesitás
4. El **presupuesto final** se actualiza en tiempo real en ARS y USD
5. Hacés click en **Descargar PDF** y obtenés el documento listo para enviar
